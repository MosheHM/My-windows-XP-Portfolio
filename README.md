# Windows XP Portfolio

A full-stack portfolio application with a Windows XP-style UI, featuring a browser-based AI chat assistant and file storage.

## Features

- Windows XP-style retro UI
- Browser-based AI chat assistant (runs in your browser using Transformers.js)
- File upload/download capabilities
- Responsive design with React 19 and TypeScript

## Quick Start

### Local Development

```bash
# Install and run the client
cd client
npm install
npm run dev
```

Visit http://localhost:5173 to see the portfolio.

## Google Cloud Deployment

For deploying to Google Cloud Platform (GKE), see [GCP_DEPLOYMENT.md](GCP_DEPLOYMENT.md) for detailed instructions.

**Quick GCP Setup:**
1. Create a GCP project and enable required APIs
2. Create a GKE cluster
3. Configure GitHub Actions secrets
4. Push to main branch for automatic deployment
